This project has only 1 aim : 

To convert a file of base 64 png image format to base 64 tiff image format

Prerequisite : 

Base 64 Encoding : 

Converting sets of 6 (0s and 1s) to 1 character of (A-z, a-z, 0-9, +, /)

Approach :

1. Decode base64_png into png
2. Convert png to tiff
3. Encode tiff into base64_tiff
